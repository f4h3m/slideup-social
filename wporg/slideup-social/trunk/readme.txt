== Description ==

Slideup Social is a simple plugin to show your facebook,twitter and instagram link when scroll down.

Here's a link of live demo [DEMO](http://demos.dopetheme.com/ignites/).

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Plugin Name screen to configure the plugin
1. To Setup your social profile you will see an option under 
`Settings > Slideup Social` where you can set your social profile for facebook,twitter and instagram.
